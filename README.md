# HX — Site

Novo site da HeadXperience Studio. Stack: Next.js 16 + TypeScript + Tailwind v4 + GSAP + Lenis.

## Comandos

```bash
npm run dev      # localhost:3000
npm run build    # production build
npm start        # serve production
npm run lint     # ESLint
```

## Estrutura

- `src/app/` — App Router pages
- `src/components/` — React components (ui, layout, home, animation)
- `src/lib/` — utilities (cn, gsap)
- `src/hooks/` — custom React hooks
- `src/styles/` — global styles, tokens
